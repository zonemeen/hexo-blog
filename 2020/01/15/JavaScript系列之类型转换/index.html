JavaScript系列之类型转换

JavaScript 是一种弱类型或者说是一门动态语言，这意味着你不用提前声明变量的类型，而是在程序运行过程中，运算符会自动将值转换为正确的类型，但也有一些情况需要强制类型转换。

以下我将介绍将各种类型的值，分别转换成数字、字符串或者布尔值（基于 ES5 规范）。

ToNumber

（1）Number()

使用 Number 函数，可以将任意类型的值转化成数值。

（2）ParseInt()

parseInt函数可用于将字符串转为整数。
